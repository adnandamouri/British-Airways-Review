# British Airways Review Project

## Tableau Dashboard

I created an interactive **Tableau dashboard** that explores **British Airways** customer feedback. You can explore the data, view ratings by aircraft, country, and service quality, and dive into trends over time.

### View the interactive dashboard here:
[Click here to explore the dashboard on Tableau Public](https://public.tableau.com/app/profile/adnan.damouri/viz/BritishAirwaysReview_17578262198200/Dashboard1)

## Description
This project analyzes **British Airways passenger feedback** to identify areas of improvement and strengths, visualized using **Tableau**.
