## Project Overview  
An in-depth **data analytics** project analyzing **British Airwaysâ€™ customer feedback** from various flight metrics. Using **Tableau**, this interactive dashboard visualizes **passenger ratings** across **aircraft types**, **countries**, and **time**, focusing on customer satisfaction in **staff service**, **food quality**, and **entertainment**.

## Key Findings  
- 253.4M reviews analyzed from **2016-2023** across **6 aircraft models**  
- **Boeing 747-400** ranked highest for **staff service** and **food quality**  
- **Passenger satisfaction** is highest in **North America** and **Europe**  
- Trends show a **steady increase** in overall ratings since 2019, with **spikes** in late 2022
